# llm-api
for host llm api easily

## usage

1. Place the necessary files.
   - .prompt
     - initial system prompt.
   - .document
     - initial document of reference for models.
   - .default
     - frontpage of faq
   - .icon
     - favicon
1. Install some libraries the model want.

1. Run
   ```sh
   python server.py model_name_any_you_want_host
   ```
